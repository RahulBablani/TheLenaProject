# User Guide

The files for this folder are as follows:
- **LenaDownload.zip** : runnable jar for project

Simply Download the zip file and extract the contents,
inside there will be 3 files:
- **README.md** : further instructions on how to run as well as how to edit for future releases
- **executable** : folder that contains executable jar in addition to external dependencies required to run the project
- **ready_for_jar** : folder that contains the files needed to make a new jar for future releases
    
# Running Lena
- open **executable** folder
- simply double click **LENA.jar**
- or in line command: *java -jar LENA.jar*
